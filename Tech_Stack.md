# HireFlow – Tech Stack

## 1. Frontend (Premium World-Class Experience)

Framework:
- React (Vite-based architecture)

3D & Motion:
- Three.js
- React Three Fiber
- Framer Motion
- GSAP (advanced timeline control)

Parallax & Scroll Effects:
- Lenis (smooth scroll)
- GSAP ScrollTrigger
- React Spring (micro interactions)

UI System:
- Custom design system (No Tailwind)
- CSS Modules / Styled Components
- Glassmorphism + Depth layering
- Dark premium theme

Animations:
- Entrance reveal animations
- Micro-interactions on hover
- Cursor-based distortion effects
- Button ripple interactions
- Dynamic gradient transitions

3D Landing Page:
- Interactive 3D globe/network graph
- Floating interview nodes
- Dynamic camera movement
- Particle background
- Motion-based hero section

Loader:
- 3D animated loader
- SVG morphing animation
- Smooth fade transitions
