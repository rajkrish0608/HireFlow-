# HireFlow – Product Requirements Document (PRD)

## 1. Product Overview

HireFlow is a premium Technical Interview-as-a-Service (TIaaS) platform designed to help startups and mid-sized tech companies outsource high-quality technical interviews to verified senior engineers.

The platform eliminates resume fraud, reduces hiring bias, and accelerates hiring cycles through structured, standardized interviews and async coding assessments.

---

## 2. Problem Statement

Tech companies face:

- Inflated or fraudulent resumes
- Lack of internal senior engineers for interviews
- Long hiring cycles (30–45 days)
- Unstructured interview processes
- High bias in evaluation
- Costly enterprise hiring tools

---

## 3. Solution

HireFlow provides:

1. On-demand technical interviewers
2. Structured evaluation rubrics
3. Recorded interviews
4. Async coding challenges
5. Skill-based matching
6. Transparent scoring system

---

## 4. Target Audience

### Primary Customers:
- Startups (20–300 employees)
- Growing tech companies
- Indian service-based companies hiring in bulk

### Secondary:
- Global remote-first companies

---

## 5. Core Features

### 5.1 Technical Interview-as-a-Service
- 60-minute deep technical interview
- Domain-matched senior engineer
- Standardized rubric
- Recording + feedback report

### 5.2 Async Coding Challenge
- Real-world coding tasks (not DSA-heavy)
- Docker sandbox execution
- Timed (30–60 mins)
- Auto code evaluation

### 5.3 Skill Graph Matching
- AI-assisted matching of interviewer expertise to job role
- Rule-based scoring initially

### 5.4 Dashboard
For HR:
- Schedule interviews
- Track candidate status
- Access reports & recordings

For Interviewers:
- Accept/decline interviews
- Submit evaluation forms
- Earnings dashboard

---

## 6. User Flow

1. HR signs up
2. Posts job role + required skills
3. Candidate assigned async challenge (optional)
4. System matches interviewer
5. Interview conducted
6. Scorecard + recommendation delivered
7. Company makes decision

---
